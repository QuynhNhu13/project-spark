# EduConnect

Nền tảng kết nối gia sư chất lượng với học sinh. Học online, luyện thi THPT Quốc Gia với AI, quản lý lớp học thông minh.

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
